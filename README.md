Akıllı Stok Yönetim Sistemi (Python)

Bu proje, işletmelerin envanter süreçlerini dijitalleştirmek ve stok takibini otomatize etmek için geliştirilmiş bir karar destek aracıdır.

 🚀 Özellikler
- *Dinamik Stok Takibi:* Satış yapıldığında mevcut envanterden otomatik düşüş sağlar.
- *Kritik Seviye Uyarısı:* Stok miktarı belirlenen eşiğin (Örn: 10 adet) altına düştüğünde kullanıcıyı uyarır.
- *Tahsilat Hesaplama:* Yapılan satışların toplam tutarını anlık olarak raporlar.

 🛠 Kullanılan Teknolojiler
- Python (Kontrol Yapıları, Fonksiyonlar, Veri Güncelleme)
